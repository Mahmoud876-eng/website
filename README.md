# 💊 Medicine Reminder Bot

This project is a **Telegram bot** that helps patients remember to take their medicines on time. It integrates with a **MongoDB database** to manage medicine data and is built with **Python**, using **Flask** and **python-telegram-bot**.

## 🚀 Features

- 🧠 Sends personalized medicine reminders based on time (morning, afternoon, night)
- 💬 Delivers notifications directly to Telegram
- 🗃️ Stores medicine data securely in MongoDB
- 🖥️ Flask server to display or manage data (optional)

## 🛠️ Tech Stack

- **Python 3**
- **MongoDB** (via `pymongo`)
- **Telegram Bot API** (via `python-telegram-bot`)
- **Flask** (for optional web interface)
